# Victor Executive Backend

Node.js + Express backend for the Victor Executive platform. It exposes REST APIs consumed by the admin portal, landing site, and mobile apps.

## Prerequisites

- Node.js 18.18+
- MySQL 8.x (database created manually, e.g. `CREATE DATABASE victor;`)

## Setup

```bash
cd backend
cp env.example .env              # update credentials and secrets
npm install
npm run dev                      # or npm start for production mode
```

On boot the server:

1. Creates/updates the database tables if they do not exist.
2. Seeds sample reference data (roles, users, airports, aircraft, flights, passengers, crew, baggage, document templates).

The API listens on `http://localhost:4000` by default. Documents generated by the backend can be downloaded from `/documents/<fileName>.pdf`.

## Authentication

Use the seeded credentials:

- `admin@victorexecutive.com / VictorAdmin!2025`
- `ops@victorexecutive.com / Operator!2025`

Obtain access and refresh tokens via `POST /api/auth/login`. Attach the access token as `Authorization: Bearer <token>` on subsequent requests.
